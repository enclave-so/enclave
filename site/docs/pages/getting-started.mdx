# Getting started [Get up and running in under a minute]

Enclave is an embedded, open-source, and non-custodial web3 wallet. It integrates with any library (Wagmi, Web3Modal, Ethers, etc.) and allows users to sign messages, send transactions, and interact with smart contracts. **It's free, with no limits or project ID required.**

[Demo](https://demo.enclave.so) | [Wallet](https://app.enclave.so)

## Manual Installation

You can try out Enclave in your dApp in under a minute.
::::steps

### Install

First, we will install `@enclave-so/sdk` as a dependency in our project.

:::code-group

```bash [npm]
npm i @enclave-so/sdk
```

```bash [pnpm]
pnpm add @enclave-so/sdk
```

```bash [yarn]
yarn add @enclave-so/sdk
```

```bash [bun]
bun i @enclave-so/sdk
```

:::

### Usage

That's it! Now you can use Enclave in your dApp. There is 2 ways to use Enclave:

#### Announce Enclave (EIP-6963) (Recommended)

```ts [main.ts]
import { provider } from '@enclave-so/sdk'

provider.announce() // EIP-6963 announce
```

:::info[EIP-6963]
Libraries like Wagmi, Web3Modal, etc., will auto-detect the provider.
:::

#### Directly use Provider(EIP-1193)

Let's use Enclave to get the list of accounts.

```ts [main.ts]
import { inject, provider } from '@enclave-so/sdk'

await inject()

// EIP-1193 provider // [!code focus]
const accounts = await provider.request({ method: 'eth_requestAccounts' }) // [!code focus]
console.log(accounts) // [!code focus]
// ['0x6B944948B5e70e4421034C4C0744A176b0bf9968'] // [!code focus]
```

:::info[EIP-1193]
You can also use the EIP-1193 provider directly.
:::

You can choose one way or use both in your dApp.
::::
