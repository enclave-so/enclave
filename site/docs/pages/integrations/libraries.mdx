# Integration with Libraries (Viem, Ethers, Web3.js)

::::steps

### Install Enclave

Install the Enclave SDK using your preferred package manager:

:::code-group

```bash [npm]
npm i @enclave-so/sdk
```

```bash [pnpm]
pnpm add @enclave-so/sdk
```

```bash [yarn]
yarn add @enclave-so/sdk
```

```bash [bun]
bun i @enclave-so/sdk
```

:::

### Setup Enclave

Add Enclave to your project setup:

```ts [main.ts]
import { inject } from '@enclave-so/sdk'

await inject()
```

### Use Enclave with Libraries

You can use Enclave with libraries like Viem, Ethers, and Web3.js. Here's an example of how to use Enclave with each:

:::code-group

```ts [viem]
import { inject } from '@enclave-so/sdk'
import { createWalletClient, custom } from 'viem'
import { mainnet } from 'viem/chains'

await inject()

const client = createWalletClient({
  chain: mainnet,
  transport: custom(provider),
})
const accounts = await client.requestAddresses()
// ['0x6B944948B5e70e4421034C4C0744A176b0bf9968']
```

```ts [ethers]
import { inject, provider as ep } from '@enclave-so/sdk'
import { ethers } from 'ethers'

await inject()

const provider = new ethers.BrowserProvider(ep)
const accounts = await provider.send('eth_requestAccounts', [])
// ['0x6B944948B5e70e4421034C4C0744A176b0bf9968']
```

```ts [web3.js]
import { inject, provider } from '@enclave-so/sdk'
import { Web3 } from 'web3'

await inject()

const web3 = new Web3(provider)
const accounts = await web3.eth.requestAccounts()
// ['0x6B944948B5e70e4421034C4C0744A176b0bf9968']
```
